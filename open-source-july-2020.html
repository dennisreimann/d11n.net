# My 4th Month of Full-Time Open Source

Last month was a bit slow code contribution-wise, but luckily this changed in July: Motivation increased rapidly and there are lots of projects to cover in this review.

# M5StackSats

I updated the BTCPay Server integration of the M5StackSats: There was an issue with the QR code generation, which lead me to pick up the project again. The UI and user flow got some improvements and I also added proper error handling in case of network errors.

A few merchants are using the M5StackSats already and there are some more things I plan to add to the integration. If you want to learn a bit more about this project, see my initial post on the BTCPay Server integration, which also contains a quick demo video.

# UIengine overhaul

Speaking of UI rebrushes: The UIengine received a major UI overhaul too. Thanks to Cos Anca the interface looks clean, fresh and sexy. He not only rebrushed the interface, but also added some new token representations: opacity, box-shadow and border styles can now also be properly displayed.

The first project I upgraded was the BTCPay Server Design System. It is a nice test case, because we allow to apply the light/dark theme to the UIengine interface as well.

# Bitcoin Donations Portal

BitcoinDevList.com was by far the project which created the most uproar this month: Matt and I launched the Bitcoin Donation Portal, which lists people working on Bitcoin and related projects. The goal is to increase the visibility of contributors to the space that are accepting donations.

I remember Matt talking about the idea for quite some time now. He recently discovered GitHub Sponsors and tweeted about some devs having a profile there. This lead to some discussion around the topic and I proposed to act on it: I envisioned some page similar to the BTCPay Server Directory, where people can enter their information. Forked the repo of this website, made some adaptions and four hours later we launched the project.

Donations and grants are important to make the development of Bitcoin as well as related projects and technologies sustainable. The donation portal has been well received, lots of people applauded us for launching something that can help the community. It also sparked some good discussions and I‚Äôm sure we and others will keep on working on this and similar projects.
